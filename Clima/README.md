## Description

iOS Weather App

-▫️Api fetched

-▫️Works with dark mode

-▫️Symbols changing with each type of weather, sunny displays sun, rainy displays rain etc.

-▫️Saves last searched city and displays its weather on startup

#### Technologies

- Cocoapods
- Open Weather API
